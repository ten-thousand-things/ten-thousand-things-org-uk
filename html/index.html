Our work

Ten Thousand Things is a software development group that loves writing code for charities, SMEs, research organisations and open source communities. We pride ourselves on working closely with partners to design and build tools that empower people.

Our values

We emphasise data sovereignty, consent and accountability as design principles. These principles serve as a platform to encourage all of us to relate to the modern web, data creation and its propagation on our own terms.

Our research

By using emerging distributed technologies, our current line of research focuses on common pool resource management and distribution. We want to discover what kind of technologies can empower people and expand the commons economy.

Projects

Dark Crystal
Dark Crystal is a protocol and prototype application making use of Shamir's secrets algorithm to build a distributed trust-based social password / private key backup mechanism. Currently being implemented as a module to bolt onto the emerging scuttlebutt ecosystem.

An open-source contribution to the Scuttlebutt community, Scuttle Invite is a module which provides an easy to use API to easily implement invite/response logic on any scuttlebutt application.

True Name
